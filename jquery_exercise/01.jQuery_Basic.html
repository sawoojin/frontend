jQuery 개요

존 레식에 의해 만들어진 JS 프레임워크(라이브러리)
기존의 document를 사용하는 방법보다 훨씬 단순하고 간결하게 사용할 수 있도록 코드 형체를 제공
애니메이션 기능, Ajax 통신, 이벤트 처리 등 폭넓게 지원함.

jQuery 사용하기


1. CDN을 통한 연결
- 온라인으로 js파일을 불러와서 실행
2. 파일 다운로드 후 연결
- jQuery홈페이지에서 최신버전 js파일 다운

jQuery 기본문법

- jQuery(selector).action()
- $(selector).action
- selecter는 css의 선택자를 의미함
- action()은 HTML에서 특정 이벤트가 발생할 때 실행하는 함수,
ex) jQuery("p"), $(p), ...
